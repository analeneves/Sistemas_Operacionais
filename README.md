Interpretador de comandos - shell

Para rodar o programa, basta abrir o terminal e usar os seguintes comandos:

$make

$./shellso

$ls -l

$ 

$ls -laR => arquivo

$wc -l <= arquivo

$cat -n <= arquivo => arquivonumerado

$cat -n <= arquivo | wc -l => numerodelinhas

$cat -n <= arquivo | wc -l => numerodelinhas

e, para sair:

$fim
